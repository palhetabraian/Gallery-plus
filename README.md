# 📸 Gallery Plus

Uma aplicação fullstack para gerenciamento e upload de imagens, permitindo organizar fotos em uma galeria online com navegação fluida e interface moderna.

---

## 🚀 Sobre o projeto

O **Gallery Plus** é uma aplicação web que permite fazer upload de imagens, visualizar fotos e organizar conteúdos de forma simples e intuitiva.  

O projeto foi desenvolvido com foco em boas práticas de desenvolvimento fullstack, integração entre front-end e back-end e experiência do usuário.

---

## 🛠️ Tecnologias utilizadas

### 🔹 Front-end
- React  
- Vite  
- Tailwind CSS  
- React Router  
- React Query  
- React Hook Form  
- Zod  

### 🔹 Back-end
- Node.js  
- Fastify  
- Upload de arquivos com multipart  
- Servir arquivos estáticos  

### 🔹 Comunicação
- Axios  

### 🔹 Deploy
- Front-end: Railway  
- Back-end: Railway  

---

## ⚙️ Funcionalidades

- 📤 Upload de imagens  
- 🖼️ Visualização de fotos  
- 📁 Organização em álbuns *(pode expandir futuramente)*  
- 🔄 Navegação entre imagens  
- 💬 Feedback de ações com toast notifications  
- 📱 Interface responsiva  
- 🔗 Integração completa entre front-end e back-end  

---

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the backend server:

```bash
pnpm dev-server
```

In another terminal, run the frontend server:

```bash
pnpm dev
```
